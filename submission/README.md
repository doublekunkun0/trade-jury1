# Submission Pack

这套文件用于把 `Trade Jury` 直接整理成可投稿材料。

文件说明：

- `SUBMISSION_ARTICLE.md`：可直接发布的长文版本
- `PROJECT_BRIEF.md`：项目简介和评分点映射
- `FINAL_PROMPT.md`：可直接提交或改写的 Prompt
- `X_THREAD.md`：X 发帖文案
- `PLANET_POST.md`：OKX 星球长文案
- `DEMO_SCRIPT.md`：30 秒录屏脚本
- `FORM_ANSWERS.md`：Google 表单填写模板
- `CHECKLIST.md`：提交前核对清单

建议使用顺序：

1. 打开演示页录屏
2. 从 `X_THREAD.md` 或 `PLANET_POST.md` 复制文案发布
3. 把发帖链接填入 `FORM_ANSWERS.md`
4. 在截止前提交表单
