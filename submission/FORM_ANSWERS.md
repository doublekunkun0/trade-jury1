# Google Form Answers

表单地址：
https://docs.google.com/forms/d/e/1FAIpQLSdhu1dQJ0oeWVIuHF8Utm34vSwjxO9q4OPydm8XuhNdDCIUlg/viewform?usp=send_form

截至我核对的公开表单内容，实际字段只有 3 个：

1. 您的 X 账号（必填）
2. 您的推文链接
3. 您的星球帖文链接

可直接填写为：

- 您的 X 账号：`@你的账号`
- 您的推文链接：`发布 X 贴文后的链接`
- 您的星球帖文链接：`发布 OKX 星球后的链接`

## 建议填写方式

如果你只发 X，不发星球：

- X 账号：填你的账号
- 推文链接：填 thread 主帖链接
- 星球帖文链接：留空或按表单实际允许情况处理

如果你同时发 X 和星球：

- 推文链接：填主帖链接
- 星球帖文链接：填长文帖链接

## 建议顺序

1. 先完成对官方主贴的点赞 / 转发 / 评论
2. 发 X thread
3. 发 OKX 星球长文
4. 把两个链接填进表单
